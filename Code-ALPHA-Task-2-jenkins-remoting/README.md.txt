Jenkins Remoting Project

Objective
Set up Jenkins controller and agent using Jenkins Remoting to distribute builds securely.

Tools Used
- Jenkins
- Java JDK
- AWS EC2 / Local VMs
- Ubuntu

Steps Implemented
1. Installed Jenkins on controller node.
2. Installed Java on agent node.
3. Connected agent using Jenkins Remoting (agent.jar).
4. Ran jobs remotely on agent.
5. Demonstrated node isolation and load distribution.

Output
- Jobs executed on remote agent successfully.
- Controller managed builds securely.
